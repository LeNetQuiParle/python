Qu'est-ce que python ?

Python est un language de programmation puissant et polyvalant, qui permet de faire baucoup de choses, dont créer et administrer un site web ou créer des applications aussi bien pour téléphones que pour ordinateurs. C'est aussi l'un des languages les plus populaires. Il a été créé par Guido van Rossum en 1989. Il devait améliorer le language de programmation ABC, utilisé pour l'éducation. Son nom vient de la troupe de comédiens "Monty Python", qu'il adorait.

Pourquoi apprendre python ?

Python est très simple, très populaire et est un language de "haut niveau", ce qui signifie qu'il simplifie la compréhension des humains plutôt que celle des ordinateurs, contrairement à d'autres languages comme javascript. Il a beaucoup d'utilisations, entre autres la robotique, le développement d'applications , l'administation d'un site web ou même l'IA et le machine learning. De plus, python peut être enrichi avec de nombreuses fonctionnalités et des éléments supplémentaires.
